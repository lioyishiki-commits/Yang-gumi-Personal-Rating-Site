# 隐私说明

Yang-gumi 采用“本地优先”设计：个人条目、评分、笔记、资源路径、访问令牌和图片文件夹路径默认只保存在运行它的电脑上。

## 会访问的外部服务

- Bangumi：搜索和读取公开条目、评分、排名、标签、简介与图片地址。
- Anime News Network、AniList、MyAnimeList、AniDB：季度条目来源核验与补充；不登录个人账户。
- 图片地址：在显示或建立缓存时下载对应公开封面。

外部服务可能按其隐私政策记录普通的网络请求信息。Yang-gumi 不需要 Bangumi OAuth，也不会写入用户的 Bangumi 账户。

## 不应提交的内容

以下目录和文件已被 `.gitignore` 排除：数据库、备份、导出、令牌、日志、本地封面、背景、季度海报、美图缓存、`.env` 与 Streamlit secrets。提交前仍应运行 `git status`，确认没有使用 `git add -f` 强制加入私人内容。

## 漏洞与敏感信息

发现安全问题或误提交的隐私数据时，请不要在公开 Issue 中粘贴令牌、数据库或私人路径；按照 [SECURITY.md](../SECURITY.md) 联系维护者并立即轮换相关令牌。

# 更新记录

## 2026-07-13

- Bangumi 公开排行改用官方 `/v0/subjects` API，按动画、漫画、小说、游戏分类读取，并升级缓存格式以清除旧网页解析产生的空缓存。
- Bangumi 跳页和跳名次改为直接读取目标区间，封面缓存改为后台执行，避免高页码按顺序补齐前面数千条。
- 今日美图只扫描用户明确选择的本机目录，并继续显示检查、识别、生成和无法读取数量。
- 新增官方排行缓存与中文路径图库的回归测试。

## 2026-07-12

- 将主站启动与只读分享彻底解耦，并增加启动器回归测试。
- 修复旧 Edge 中输入框、数字框、展开栏和上传区出现白边/白框的问题。
- 今日美图支持有上限的多级子目录递归扫描。
- 数据管理增加 `.db` 加载入口；新增可选保存数据库并校验完整性的 Windows 卸载程序。
- 更新新电脑部署说明和 Dark Reader 旧浏览器兜底步骤。

## 2026-07-06

- 发布可独立部署的本地版本。
- 增加 Windows 安装与启动脚本。
- 补充旧版 Chromium Edge 的前端兼容处理。
- 完善数据目录隔离、隐私说明和迁移文档。
- 修正页面在浏览器缩放时的卡片布局。
